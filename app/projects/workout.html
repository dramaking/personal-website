About

The Workout Planner targets users who go to the gym regularly and have the habit to keep workout records. The UX design is minimalist, smooth and straight-to-the-point to highlight functionality. The kanban style board design gives users a familiar feeling and enable them to pick up right from the beginning.

Development

This is the first single page application I built based on React library. The workout, exercise, note and set provide the full functionality to create, edit and delete. The mobile version of this application is under development to provide a truly practical product.
