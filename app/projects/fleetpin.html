Fleetpin Mobile App

Fleetpin Mobile App provides a more efficient way to manage the vehicle information for small business. It is developed with React Native which targets both iOS and Android platform.

View App

Client

Fleetpin Ltd

My role

Mobile Development

Year

2017

#React Native #iOS #Android #ES6

Background

Fleetpin is a New Zealand tech startup providing cloud based fleet management solution for customers. By installing GPS on clients' vehicles and colleting data that is sent through these devices, Fleetpin provides a web application that can show the full story of where the vehicles have been and what their current status is.

Challenge and Approach

The client asked for a product that can transfer the most popular and basic features from the web application to the mobile application. The design is made sleek and minimalist delibrately for users to focus on the information at hand. UI elements are carefully selected to incorporate with the standard mobile theme and current trend, so that content can go first.

The development of the app takes into consideration of both iOS and Android plaiforms. We manage to keep the UI and code base as cohesive as possible so that the project will be more maintainable for future versions of the app.

Highlights

From design, development to deployment, it only took 2 months for the first version of the Fleetpin app to be released on App Store. With a few simple clicks on their mobiles, customers can view live vehicle information updated every 20 seconds from the cloud.
